= SrcLab BOM

BOM of srclab projects

== Getting started

Maven:

[source,xml]
----
<dependency>
    <groupId>xyz.srclab.bom</groupId>
    <artifactId>srclab-bom</artifactId>
    <version>1.0.0</version>
</dependency>
----

Gradle:

[source,groovy]
----
api platform("xyz.srclab.bom:srclab-bom:1.0.0")
----
